# 期貨下載器

台灣期貨交易所 ZIP 檔案自動下載工具

## 功能

- 自動爬取 taifex.com.tw 網站
- 多執行緒下載（5線程）
- 自動跳過最新檔案，下載歷史檔案
- 內建 Windows 排程設定

## 使用方法

### 1. 啟動程式

```bash
python 期貨下載器.py
```

### 2. 選單操作

- **1** - 立即下載
- **2** - 設定自動排程（週日/工作日/每日）
- **3** - 查看排程狀態
- **4** - 移除排程
- **0** - 離開

### 3. 排程執行

設定排程後，系統會自動執行：
```bash
python 期貨下載器.py download
```

## 設定

檔案下載路徑：`E:/rpt`

修改 `CONFIG['path']` 可變更下載位置。
