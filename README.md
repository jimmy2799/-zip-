# 期貨ZIP檔案下載器

這個專案包含一個自動化工具，用於下載台灣期貨交易所的歷史交易資料ZIP檔案。

## 功能特色

- 📥 自動爬取期貨交易所網站
- 📦 下載歷史交易資料ZIP檔案
- ⏰ 支援Windows任務排程器自動執行
- 🔄 多執行緒下載，提高效率
- 📁 自動管理下載目錄

## 檔案說明

### 主要腳本
- `爬期貨ZIP檔案_優化版.py` - 主要的下載腳本
- `排程設定_自動執行.py` - Windows任務排程器設定腳本

### 配置

預設下載目錄：`E:/rpt`

如需更改下載目錄，請修改 `爬期貨ZIP檔案_優化版.py` 中的 `CONFIG['path']` 設定。

## 使用方法

### 手動執行
```bash
python 爬期貨ZIP檔案_優化版.py
```

### 自動排程設定
```bash
python 排程設定_自動執行.py
```

執行後會進入互動式設定介面，可以選擇：
- 每周日自動執行
- 工作日自動執行
- 自定義排程設定

## 技術細節

- 使用 `requests` 進行HTTP請求
- 使用 `BeautifulSoup` 解析HTML
- 使用 `concurrent.futures` 實現多執行緒下載
- 支援Windows任務排程器自動化

## 注意事項

1. 確保目標目錄有寫入權限
2. 建議在非交易時間執行，避免影響網路效能
3. 可根據需要調整下載時間和頻率
